# Prompt 08 - Theme, Responsive, and UI Polish

Gunakan prompt ini untuk mengerjakan **Stage 8** dari proyek portfolio website. Fokus pada light/dark mode, responsive behavior, dan penyempurnaan UI.

## Prompt

Sempurnakan portfolio website ini menggunakan pendekatan **Multi-Stage Prompting**. Pada stage ini, fokus pada kualitas visual dan pengalaman pengguna lintas device.

## Project Context

- Gaya visual: modern clean
- Website harus responsif untuk laptop, tablet, dan mobile
- Mendukung light mode dan dark mode
- Menggunakan `Tailwind CSS` dan `Framer Motion`
- Hero sudah memiliki visual keyboard image sequence

## Stage Goal

Meningkatkan kualitas tampilan dan pengalaman pengguna agar website terasa matang, konsisten, dan profesional.

## Multi-Stage Instructions

### Step 1 - Audit UI State

- Tinjau seluruh halaman publik dan admin.
- Identifikasi area yang belum konsisten secara spacing, hierarchy, warna, dan interaction.

### Step 2 - Implement Theme System

- Bangun atau rapikan light/dark mode yang konsisten.
- Pastikan theme tidak hanya mengganti background, tetapi juga menjaga contrast, shadow, dan readability.

### Step 3 - Improve Responsiveness

- Pastikan layout bekerja baik di desktop, tablet, dan mobile.
- Rapikan spacing, grid, navigation behavior, dan visual balance pada setiap breakpoint utama.

### Step 4 - Polish Motion and Details

- Gunakan motion yang halus dan relevan.
- Rapikan state hover, transition, loading feedback, dan detail antarmuka lainnya.

### Step 5 - Verify

- Pastikan tampilan terasa konsisten di seluruh halaman.
- Pastikan theme switch dan responsive behavior berjalan baik.
- Ringkas peningkatan UI/UX yang dilakukan.

## Expected Output

- Theme system yang matang
- Website responsif di berbagai device
- UI lebih rapi, hidup, dan konsisten

## Constraints

- Hindari animasi yang terlalu ramai
- Jaga arah visual tetap modern clean
- Fokus pada kualitas, bukan menambah fitur baru
